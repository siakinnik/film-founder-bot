# FilmFounder Bot 🎬

![Status](https://img.shields.io/badge/status-alpha-orange)
![License](https://img.shields.io/badge/license-MIT-blue)

AI-powered Telegram bot to find movies by description using Google Gemini.

## 🛠 Project Status: Alpha (Work in Progress)

This project is currently under active development. The core infrastructure is implemented, and I am currently working on finalizing the AI search logic and localization strings.

### 📍 Current Progress

- [x] **Core Architecture:** Telegraf bot setup with session management.
- [x] **Database:** SQLite integration for user settings and usage limits.
- [x] **Memory Management:** Automated session cleanup (Memory Cleaner).
- [ ] **AI Logic:** Full integration of Gemini 2.5 Flash Lite search capabilities.
- [ ] **Localization:** English (Primary), Russian, and German (Current DE files are placeholders).
